# Instrucciones para Ejecutar el Proyecto

Este proyecto es un formulario de registro de usuarios con una interfaz web desarrollada en Java.

## Pasos para Ejecutar

1. **Descarga del Proyecto:**
   - Ingresa al repositorio del proyecto en [GitHub](https://github.com/diegocano2001/Creacion_Users_Java_Web).
   - Haz clic en el botón verde "Code" en la parte superior y selecciona "Download ZIP".
   - Descomprime el archivo descargado en tu computadora.

   **o**

   - Si estás familiarizado con Git, puedes clonar el repositorio con el siguiente comando:
     ```
     git clone https://github.com/diegocano2001/Creacion_Users_Java_Web
     ```

2. **Requisitos:**
   - Asegúrate de tener Java instalado en tu computadora.
   - Necesitarás un servidor web para ejecutar la aplicación. Recomendamos Apache Tomcat.

3. **Configuración:**
   - Abre el proyecto en tu entorno de desarrollo Java (por ejemplo, Eclipse, IntelliJ).

-Si no tiene Tomcat descargado puedes seguir estos pasos adisionales:
		**Pasos Detallados:**

A. **Configuración del Servidor Web (Tomcat):**
   - Descarga e instala Tomcat desde [Apache Tomcat](http://tomcat.apache.org/).
   - Abre tu entorno de desarrollo Java (por ejemplo, Eclipse, IntelliJ).

B. **Despliegue del Proyecto:**
   - Importa tu proyecto Java en tu entorno de desarrollo.
   - Configura tu proyecto para usar Tomcat como servidor de aplicaciones.
   - Compila tu proyecto para generar el archivo WAR.

C. **Instalación en Tomcat:**
   - Copia el archivo WAR generado en el directorio `webapps` de la instalación de Tomcat.
   - Inicia o reinicia tu servidor Tomcat para desplegar la aplicación.


4. **Ejecución:**
   - Configura tu servidor web (como Tomcat) y asegúrate de que esté en ejecución.
   - Despliega el proyecto en tu servidor web.

5. **Acceso a la Aplicación:**
   - Abre tu navegador web y visita http://localhost:8080/PruebaTecnicaJava/
